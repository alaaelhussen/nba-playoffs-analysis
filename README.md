## 2023-24 NBA Playoffs Data Analysis
Analyzed playoff performance across 16 teams using Python. Built interactive dashboards with dropdown menus to explore team progression through rounds, semifinals, and finals. 
## What I Built: 
- ** Player stats dashboard** - top 10 players by points, assists, rebounds, steals, and blocks with interactive dropdown
- ** Conference standings** - top 5 teams per conference with toggleable stat filters
- ** Finals game log analysis** - Nuggets vs Heat home/away performance breakdown
- ** Playoff map ** - all 16 playoff teams plotted geographically by how far they advanced
## Tools:
Python, pandas, Plotly, ipywidgets, matplotlib
## Data source
Basketball Reference (scraped live via pd,read_html)
